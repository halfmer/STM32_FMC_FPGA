# Import To Pango

Existing Pango project:

```text
prj/project/project.pds
```

The existing project already targets:

```text
Family: Logos
Device: PGL12G
Speed:  -6
Package: LPG144
```

## Import Steps

1. Open `prj/project/project.pds` in Pango Design Suite.
2. The Verilog files are already listed in the project:
   - `../../rtl/fifo.v`
   - `../../rtl/my_fifo.v`
   - `../../rtl/fmc.v`
   - `../../rtl/fifo_and_fmc.v`
   - `../../rtl/pll_50m_to_130m_bb.v`
   - `../../rtl/body.v` as top module `body`
3. `timing.fdc` is already imported by the Pango project.
4. Generate the PLL IP:
   - module name: `pll_50m_to_130m`
   - input clock: 50 MHz
   - output clock: 130 MHz
   - include `locked`
5. Remove or exclude `rtl/pll_50m_to_130m_bb.v` after the real PLL IP is added.
6. Add board pin constraints using `constraints/pin_constraints_template.txt`
   as a checklist.
7. Compile and review:
   - unresolved black boxes
   - width truncation warnings
   - inferred RAM/FIFO mapping
   - unconstrained clocks and paths
